# Проект p/template_metaproject/module

## Что это

Template Module: чистый пример Chatium-модуля для metaproject. Модуль находится
рядом с `p/template_metaproject/core` и показывает минимальную интеграцию с core
broker через `runAppFunction`.

## Инварианты

- `PROJECT_ROOT` модуля: `p/template_metaproject/module`.
- Core broker target app: `p/template_metaproject/core`.
- В модуле нет ссылок на конкретные внешние интеграции.
- Broker-вызовы идут только через server-side wrapper `lib/broker/coreBrokerClient.lib.ts`.
