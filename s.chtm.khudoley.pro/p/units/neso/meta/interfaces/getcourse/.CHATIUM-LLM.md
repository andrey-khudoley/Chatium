# Проект p/units/neso/meta/interfaces/getcourse

## Что это

GetCourse Interface: Chatium-модуль NESO meta для событий GetCourse. Модуль
находится рядом с `p/units/neso/meta/core` и интегрируется с core broker через
`runAppFunction`.

## Инварианты

- `PROJECT_ROOT` модуля: `p/units/neso/meta/interfaces/getcourse`.
- Core broker target app: `p/units/neso/meta/core`.
- Модуль публикует `getcourse.raw_event.accepted@1`.
- Broker-вызовы идут только через server-side wrapper `lib/broker/coreBrokerClient.lib.ts`.
