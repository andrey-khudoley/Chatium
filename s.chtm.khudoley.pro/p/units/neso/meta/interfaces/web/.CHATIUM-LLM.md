Scaffold Chatium-модуля `p/units/neso/meta/interfaces/web` для публичной checkout-формы NeSo Meta.

## Критические правила

- Runtime checkout-flow пока не реализован; не считать template-главную финальной формой.
- Перед изменением кода сначала обновлять `docs/spec/spec.md`.
- Web flow должен идти через broker: `web.checkout.submitted@1` → GetCourse → `getcourse.order.created@1` → WebSocket `payment_ready`.
- Vue не импортирует server-only слои.
