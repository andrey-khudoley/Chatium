## Ключевые точки входа

| Файл                          | Назначение                                            |
| ----------------------------- | ----------------------------------------------------- |
| `api/ingest/index.ts`         | Публичный приём снимков (POST, без авторизации, CORS) |
| `api/embed/script.ts`         | Встраиваемый JS-скрипт для GetCourse-страниц          |
| `web/admin/index.tsx`         | Роут-обёртка админки                                  |
| `pages/AdminPage.vue`         | Интерфейс админки (лист, фильтры, detail, тумблер)    |
| `tables/diagnostics.table.ts` | Heap-таблица обращений                                |
| `lib/settings.lib.ts`         | Тумблер вкл/выкл приёма                               |
