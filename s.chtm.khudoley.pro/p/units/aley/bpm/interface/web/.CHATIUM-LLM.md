# .CHATIUM-LLM

Проект: BPM Терминал — веб-интерфейс личной BPM-системы «FLOW».

## Что это

Single-page Vue SPA, монтируемое Chatium-платформой. SSR-entrypoint: `index.tsx`. Корневой компонент: `pages/AppShell.vue`.

Стадия 1 — mock-first (без Heap, tables, repos). Все данные из `shared/mocks/*.mock.ts`.

## Ключевые файлы

- `docs/spec/spec.md` — спецификация (источник истины)
- `README.md` — структура, состояние, changelog
- `shared/store.ts` — реактивное состояние (`// @shared`)
- `shared/types/index.ts` — все типы (`// @shared`)
- `config/project.tsx` — SCREEN_TITLES, SCREEN_CODES
- `pages/AppShell.vue` — корневой shell, инициализация store

## Инварианты

- Vue-компоненты импортируют только из `shared/*` (`// @shared`)
- Не импортировать `tables/`, `repos/`, `lib/` в компоненты
- `ctx`, `app` — глобальные, не импортировать
- Логирование только в `index.tsx` через `ctx.account.log()`
- `@app/html-jsx` — с `// @ts-ignore` перед импортом

## Навигация

Активный экран — `store.screen`. Переключение через `go(screen)` из `shared/store`.
Детали — `openDetail(type, id)` / `closeDetail()`.
