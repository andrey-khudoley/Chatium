# chatium-price

Одностраничный проект-витрина: показывает прайс-лист тарифов Chatium.

## Текущее состояние

- Единственный роут `/` ([index.tsx](index.tsx)) — статическая страница прайс-листа, рендер на сервере через `@app/html-jsx` (без Vue и клиентского бандла).
- 4 тарифа: Бесплатный / Старт / Pro / Макс. Карточка «Макс» — акцентная, с градиентным свечением.
- Дизайн снят с [chatium.ru/pricing](https://chatium.ru/pricing): шрифт Inter, белые карточки со скруглением и мягкой тенью, зелёные галочки в списке фич.
- Тарифы и цены — из актуального макета, вынесены в [shared/pricing.ts](shared/pricing.ts). CSS — в [shared/styles.ts](shared/styles.ts).
