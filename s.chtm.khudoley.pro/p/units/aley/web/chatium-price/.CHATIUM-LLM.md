# chatium-price — указатель для LLM

Проект-витрина: одна страница `/` с прайс-листом тарифов Chatium.

- **Что это:** статический прайс-лист (4 тарифа), SSR через `@app/html-jsx`, без Vue.
- **README:** [README.md](README.md) — состояние, футер, changelog.
- **Документация:** [docs/](docs/) — architecture.md, api.md, data.md; журнал в [docs/LLM/](docs/LLM/).
- **Источник правды по тарифам и ссылкам кнопок:** [shared/pricing.ts](shared/pricing.ts). CSS — [shared/styles.ts](shared/styles.ts). Роут — [index.tsx](index.tsx).
- **Футер:** только копирайт, юр-ссылок нет (документов у проекта не будет).
