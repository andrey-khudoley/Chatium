# API chatium-price

Проект не имеет собственных API-эндпоинтов. Единственный публичный роут — страница.

## `GET /`

- **Файл:** [index.tsx](../index.tsx), `indexPageRoute = app.html('/', …)`.
- **Тип:** HTML-страница (SSR через `@app/html-jsx`).
- **Авторизация:** не требуется — публичная витрина.
- **Ответ:** полный HTML-документ прайс-листа (4 тарифа + футер).
- **Побочные эффекты:** `ctx.account.log(...)` при каждом рендере.

Межроутовых ссылок по URL в проекте нет. Реестр путей — [config/routes.tsx](../config/routes.tsx).
