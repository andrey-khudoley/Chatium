# Данные chatium-price

Проект не использует Heap/БД. Все данные статичны и лежат в [shared/pricing.ts](../shared/pricing.ts).

## `Tariff`

| Поле          | Тип                        | Описание                                              |
| ------------- | -------------------------- | ----------------------------------------------------- |
| `id`          | `string`                   | Уникальный ключ тарифа (для `key` в списке).          |
| `name`        | `string`                   | Название тарифа.                                      |
| `price`       | `string`                   | Актуальная цена с разрядами, без символа валюты.      |
| `oldPrice`    | `string \| null`           | Старая (зачёркнутая) цена; `null` — акции нет.        |
| `period`      | `string`                   | Период под ценой (напр. «в месяц»).                   |
| `ctaLabel`    | `string`                   | Подпись кнопки.                                       |
| `ctaVariant`  | `'primary' \| 'secondary'` | Стиль кнопки (тёмная / светлая).                      |
| `features`    | `string[]`                 | Список фич в порядке отображения.                     |
| `featured`    | `boolean`                  | Акцентная карточка с градиентным свечением.           |

Константы: `CURRENCY` (символ валюты), `TARIFFS` (массив тарифов), `HERO` (копирайт и
ссылки кнопок хедера), `TARIFF_CTA_HREF` (общая ссылка кнопок тарифов → регистрация).
