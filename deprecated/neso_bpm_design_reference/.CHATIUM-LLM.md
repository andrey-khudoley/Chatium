# Проект NeSo BPM

## Что это

Chatium BPM workspace с Vue-интерфейсом command center, каталогом demo-сценариев и reusable BPM-компонентами.

## Где искать детали

- Техническая информация: `README.md`
- Общие типы для Vue-компонентов BPM (шаринг с `components/index.ts`): `shared/bpmVueExportedTypes.ts`
- Demo-состояния и копирайтинг интерфейса: `shared/bpmScenarioDemo.ts`, `shared/bpmI18n.ts`
- Токены тем и глобальные стили: `shared/themeCatalog.ts`, `shared/themeStyles.ts`
- История диалогов: `docs/LLM/` (при появлении файлов)

## Критические правила

- Не хардкодить URL, использовать `config/routes.tsx`.
- Документацию обновлять при любых изменениях структуры/логики.
