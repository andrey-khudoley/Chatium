# GetCourse Docs (Геткурс документация)

## Назначение
Публичный веб-интерфейс для просмотра документации GetCourse Tech API. Данные берутся из OpenAPI-спецификации (файл в проекте), отображаются в виде разделов по тегам и спойлеров по операциям. Без авторизации, без Heap.

## Важно
- Платформа: Chatium. Серверная часть управляется платформой.
- Стек фиксирован платформой, новые зависимости не добавляем.
- Деплой происходит автоматически после пуша.
- Авторизация не используется: страница документации публичная.

## Текущее состояние
- Один роут: главная страница index.tsx (SSR).
- Данные: OpenAPI-файл data/openapi-schema.json читается через readWorkspaceFile из @start/sdk при каждом запросе, нормализуются в lib/openapi.lib.ts.
- UI: Vue-страница DocsPage.vue с поиском, навигацией по разделам (тегам), спойлерами по операциям; компоненты ApiSection, ApiOperation, SchemaView, AppFooter.
- Стили: светлая тема, CSS-переменные, Tailwind; мобильная адаптация обязательна.
