# Проект p/getcourse_docs (Геткурс документация)

## Что это
Публичная страница документации GetCourse Tech API. Один роут, без авторизации, без Heap. Данные из OpenAPI-файла в проекте, отображение: разделы по тегам, спойлеры по операциям, схемы запроса/ответа.

## Где искать детали
- Общее описание: README.md
- Архитектура и слои: docs/architecture.md
- Импорты: docs/imports.md
- История диалогов: docs/LLM/

## Критические правила
- Не хардкодить URL, использовать config/routes.tsx.
- Чтение OpenAPI: readWorkspaceFile(ctx, "p/getcourse_docs/data/openapi-schema.json") из @start/sdk.
- Документацию обновлять при любых изменениях структуры/логики.
